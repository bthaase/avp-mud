Available Upgrades for M4A3 Pistol:
1) Tactical Light
2) Stabilizer
3) Silencer
4) Laser Sight

Upgrade Information: 

Name: Tactical Light (Done)
Price: 100 Exp
Weight: 1 Pound
Description:
Activated by USEing the gun, the TAC Light acts as
a gun-mounted flashlight until the battery depletes,
at which point it shuts off to recharge.

Name: Stabilizer (Done)
Price: 400 Exp
Weight: 1 Pound
Description:
The Stabilizer vents the gas from the shot with a
L-Shaped attachment to the barrel, so it counters
out most of the weapons recoil. Improves accuracy
by upwards of 20%.

Name: Silencer (Done)
Price: 300 Exp
Weight: 1 Pound
Description:
Allows the M4A3 to fire without making a sound at
all. Good against aliens who have amazing hearing.

 Name: Laser Sight (Done)
Price: 400 Exp
Weight: 1 Pound
Description:
Fits the M4A3 with a laser sight that can increase
your combat range by an extra 10 meters, and makes
it 10% more accurate at range.

Available Upgrades for M240 Flamethrower:
1) Thermal Cooling
2) Napthal Injection
3) Compressed Tanks
4) Motion Tracker

Upgrade Information: 

Name: Thermal Cooling (Done)
Price: 300 Exp
Weight: 1 Pound
Description:
Uses liquid nitrogen cycled around the firing barrel to
keep the M240 from overheating so much during firing.
Allows the weapon to fire a full round earilier due to
the cooling effect. -1 to AP.

Name: Napthal Injection (Done)
Price: 400 Exp
Weight: 1 Pound
Description:
Injects a special mixture of napthal into the stream,
increasing the result damage by 20%. Gives the M240
that extra kick it needs in combat.

Name: Compressed Tanks (Done)
Price: 600 Exp
Weight: 2 Pound
Description:
Allows the M240 to hold twice as much fuel in the same
space as usual, doubling the amount of shots before
having to reload the weapon. 

Name: Motion Tracker (Done)
Price: 400 Exp
Weight: 1 Pound
Description:
Provides a mounted motion tracker on the stock of the
M240 Flamethrower, so Marines aren't left in the dark
(so to speak) while bughunting.

Available Upgrades for M90 Minigun:
1) Stabilizer System
2) Flood Lamp
3) Spinup Mechanism

Upgrade Information: 

Name: Stabilizer System (Done)
Price: 800 Exp
Weight: 1 Pound
Description:
Uses a special modification to the barrel to allow the
gas from firing to vent out of the nozzel sides, adding
a downward kick to the gun to counter the recoil. Allows
you to fire the weapon up to 10 meters farther.

Name: Flood Lamp (Done)
Price: 200 Exp
Weight: 1 Pound
Description:
 Activated by USEing the gun, the Flood Lamp acts as a
weapon-mounted flashlight until the battery depletes,
at which point it shuts off to recharge. Unlike the
TACLights, the lamp lasts much longer.

Name: Spinup Mechanism (Done)
Price: 600 Exp 
Weight: 1 Pound
Description:
An automatic system that keeps the barrel of the M90
close to the necessary firing time constantly. It 
allows the marines to respond faster, providing a
full -1 to AP on the M90 Minigun.

% Design Information :: Object Design                                  %
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

So far, the required types of upgrades include the following:
 Attached Lights
 Weapons (Generally Grenade Style)
 Stat Mods (For the weapon)
 Flag changes (For the weapon)
 Affect Mods (For the player)
... And several others I'm still defining ...

Item Type :: ATTACHMENT
 v0 - Attachment Type
 v1 - Weapon Vnum
 For LIGHTS:
  v2 - Charge
  v3 - Max Charge
  v4 - State (ON/OFF)
 For WEAPONS: (Grenade Style)
  v2 - Ammo Code
  v3 - Range (Rooms)
  v4 - Reset Time (Drains from main AP)
 For ACCURACY: 
  v2 - Same room % modifier
  v3 - Ranged % modifier
  v4 - Distance modifier
  v5 - Weight decrease
 For SILENCER:
  v2 - Dampening level
 For DAMAGE:
  v2 - Damage % Modifier
  v3 - Damage Bonus (Flat rate)
  v4 - Weight modifier
 For TRACKER:
  v2 - State (ON/OFF)
 For VISION:
  v2 - Darkscan modifier
  v3 - Provides Infrared?
 For AMMO:
  v2 - Ammo Modifier (*X)
  v3 - Weight decrease
 For FIRERATE:
  v2 - Fire Rate decrease
  v3 - Weight decrease
 For SPEEDLOAD:
  v2 - AP cost to reload
